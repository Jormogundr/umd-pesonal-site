CIS525 - Web Technology

This course is being taught by Professor Jinhua Guo. It covers design and implementation of technologies for web-based applications, including:

  • HTML & CSS
  • Javascript, jQuery, JSON
  • PHP, Node.js
  • React
  • Security
  • AJAX, Apache, MySQL, and more

The course is very project focused, including a large-scale term project. Check out the syllabus for more information.
